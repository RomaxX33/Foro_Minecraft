PhpParser\Node\Expr\CallLike::getArg ( string  $name,

Retrieves a specific argument from the raw arguments.

Returns the named argument that matches the given $name, or the positional (unnamed) argument that exists at the given $position, otherwise, returns null for first-class callables or if no match is found.
